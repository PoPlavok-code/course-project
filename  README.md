#  Курсовая работа: Приложение для анализа банковских операций

## 📋 Описание

Приложение для анализа банковских транзакций из Excel-файла. Программа загружает данные о транзакциях, позволяет фильтровать их по различным критериям и генерировать отчёты.

### Реализованные задачи:

1. **Веб-страницы** — генерация JSON-ответов для главной страницы
2. **Сервисы** — сервис «Инвесткопилка» (округление трат)
3. **Отчёты** — отчёт о тратах по категориям

📖 Описание функционала
1️ Главная страница (пункт 1)
Генерирует JSON-ответ со следующими данными:
Приветствие — зависит от времени суток (утро/день/вечер/ночь)
Карты — список карт с общей суммой расходов и кешбэком
Топ-5 транзакций — по сумме платежа
Курсы валют — USD, EUR (из настроек)
Цены акций — AAPL, AMZN, GOOGL, MSFT, TSLA

Пример JSON-ответа:
{
  "greeting": "Добрый день",
  "cards": [
    {
      "card": "1234",
      "total_amount": 15000.50,
      "cashback": 150.00
    }
  ],
  "top_transactions": [...],
  "currency_rates": [
    {"currency": "USD", "rate": 92.5},
    {"currency": "EUR", "rate": 98.3}
  ],
  "stock_prices": [
    {"stock": "AAPL", "price": 175.5}
  ]
}

2️⃣ Инвесткопилка (пункт 2)
Сервис рассчитывает сумму, которую удалось бы отложить в «Инвесткопилку» за счёт округления трат.
Как работает:
Пользователь вводит месяц (например, 2024-01)
Пользователь выбирает лимит округления: 10, 50 или 100
Каждая транзакция округляется вверх до ближайшего лимита
Разница между округлённой и фактической суммой попадает в копилку

Пример:
Трата: 1712 ₽
Лимит: 50 ₽
Округление: 1750 ₽
В копилку: 38 ₽

3️⃣ Простой поиск (пункт 3)
Поиск транзакций по описанию или категории.
Пример:
Введите строку для поиска: Супермаркеты

Найдено 15 транзакций:
- 2024-01-15: Пятёрочка
- 2024-01-18: Магнит
...

4️⃣ Отчёт по категории (пункт 4)
Генерирует отчёт о тратах по заданной категории за последние 3 месяца.
Особенности:
Декоратор @save_report() автоматически сохраняет результат в JSON-файл
Если имя файла не указано — используется имя функции
Отчёт сохраняется в формате JSON
Пример:
Введите категорию: Супермаркеты

Траты по категории 'Супермаркеты':
         Дата операции  Сумма операции    Описание
0         2024-01-15        1500.50   Пятёрочка
1         2024-01-18        2300.00      Магнит
...

📝 Примеры использования

Решение проблем

Решение: Запускайте программу из корня проекта:

cd course-project

Решение: Убедитесь, что файл operations.xlsx находится в папке data/.
Ошибка: JSONDecodeError в user_settings.json
Решение: Проверьте, что файл user_settings.json содержит валидный JSON:

{
  "user_currencies": ["USD", "EUR"],
